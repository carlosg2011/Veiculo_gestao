## 📌 Sobre o Projeto

A **VehicleGuard API** é uma API REST que será desenvolvida em C# com o objetivo de gerenciar veículos em um contexto semelhante ao de seguradoras ou associações veiculares.

O projeto tem como foco a construção de uma aplicação backend estruturada, aplicando boas práticas de desenvolvimento, organização em camadas e segurança.

---

## 🎯 Proposta

Desenvolver uma API que permita:

- Gerenciamento de veículos  
- Controle de status  
- Autenticação segura com JWT  
- Persistência em banco de dados  
- Tratamento de erros HTTP
